Task 1: Pseudocode an Everyday Task

Pseudocoding Gardening

  1. Find a place to garden.
  2. Find a spot with good soil.
  3. Gather seeds, garden tools, and fertilizer.
  4. Gather water.
  5. Prepare the soil.
    1. Remove weeds.
    2. Dig holes for plants.
    3. Spread fertilizer.
  6. Plant the seeds
    1. Water the seeds.
    2. Cover the seeds with soil.
    3. Wait for plant to grow.

Task 2: Pseudocode a Computer Task

Pseudocoding a Hangman Game

  1. Print game board
  2. Accept user input
  3. Check user input against hidden solution
    1. If user input is correct, reveal solution
    2. Else print additional part of hangman
      1. If hangman is fully drawn, user loses
      2. Else print additional hangman part
